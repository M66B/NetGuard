NetGuard


Short description
-----------------
免root防火墙


Long description
----------------
NetGuard是一款简洁的网络权限管理应用 - 无需root权限.

可独立允许/阻止软件访问WiFI和/或移动网络.

阻止应用访问网络有利于:

- 节省您的数据流量
- 节电
- 保护您的隐私

特性:

- 无需root权限
- 简单易用
- 免费
- 开源
- 无额外耗电
- 无远程服务器连接
- 无用户追踪或用户分析
- 无广告
- 支持IPv4/IPv6 TCP/UDP
- 控制系统应用和用户应用

NetGuard自身不要求网络权限.

NetGuard也许是安卓上第一款免费开源且无需root权限的防火墙应用.
